# Nanu {not another nsp updater} for Termux

## Prerequisites
- [Termux](https://f-droid.org/en/packages/com.termux/)

- Copy paste the following command in Termux:
```sh
pkg update && pkg upgrade && pkg install git && git clone https://github.com/alfhashut/NspUpdater_Termux_Script.git && cd NspUpdater_Termux_Script && bash NspUpdaterÄ°nstaller
```

## Usage

What to do next?

1. Copy your games, updates, and keys to the `NspUpdater_Termux_Script` folder.

2. You can launch the program in 2 ways:
```sh
python Menu.py

# OR

NspUpdaterRun
```
